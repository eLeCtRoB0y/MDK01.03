# Flutter Counter App (Разбор практических заданий)

Учебное Flutter-приложение, демонстрирующее управление состоянием (`StatefulWidget`), динамическую стилизацию текста, валидацию диапазонов и отслеживание истории изменений.

---

##  Как запустить проект

1. **Создайте новый Flutter-проект**:
   ```bash
   flutter create counter_app
   cd counter_app
   ```
2. **Замените код** в файле `lib/main.dart` на код реализации.
3. **Запустите приложение** в эмуляторе или браузере:
   ```bash
   flutter run
   ```

---

##  Подробный разбор выполненных заданий

### 🔹 Задание 1: Базовая структура приложения
* **Архитектура**: Приложение реализовано с использованием `StatefulWidget` (классы `CounterApp` и `_CounterAppState`). Это позволяет динамически обновлять интерфейс при изменении данных.
* **Состояние**: Внутри `_CounterAppState` определена приватная переменная счетчика:
  ```dart
  int _counter = 0;
  ```

### 🔹 Задание 2: Отображение счетчика
* **Виджет**: Для вывода значения используется стандартный виджет `Text`.
* **Стилизация**: Текст сделан крупным и выделенным с помощью параметров `fontSize: 72` и `fontWeight: FontWeight.bold`.

### 🔹 Задание 3: Кнопки управления
* **Компоновка**: Три кнопки управления (`-1`, `Сбросить`, `+1`) обернуты в виджет `Row` с выравниванием по центру (`MainAxisAlignment.center`).
* **Интерактивность**: Внутри обработчиков нажатия (`onPressed`) изменение данных обернуто в `setState()`. Это заставляет Flutter мгновенно перерисовывать экран.

### 🔹 Задание 4: Визуальная обратная связь
* **Динамический цвет**: Написан метод `_getCounterColor()`, который возвращает цвет текста в зависимости от значения:
  * Положительное число $\rightarrow$ **зеленый** (`Colors.green`)
  * Отрицательное число $\rightarrow$ **красный** (`Colors.red`)
  * Ноль $\rightarrow$ **синий** (`Colors.blue`)
* **Информационное сообщение**: Под счетчиком выводится текстовое пояснение текущего состояния числа («Положительное число», «Отрицательное число» или «Ноль»).

### 🔹 Задание 5: Дополнительные функции
* **Ограничение диапазона**: В логику изменения счетчика добавлено условие: `if (newValue >= -10 && newValue <= 10)`. Если значение выходит за рамки $[-10; 10]$, состояние не обновляется.
* **История**: Добавлена переменная `int? _previousCounter`. Перед каждым изменением в нее сохраняется текущее значение, которое затем отображается на экране в виде серого текста.

##  Спецификация методов и переменных


| Компонент | Тип | Описание |
| :--- | :--- | :--- |
| `_counter` | `int` | Текущее значение счетчика. |
| `_previousCounter` | `int?` | Предыдущее значение (может быть `null` при старте). |
| `_updateCounter(int delta)` | `void` | Изменяет шаг счетчика и проверяет границы $[-10; 10]$. |
| `_resetCounter()` | `void` | Сбрасывает счетчик в `0` с сохранением истории. |
| `_getCounterColor()` | `Color` | Определяет цвет шрифта по знаку числа. |
| `_getCounterMessage()` | `String` | Возвращает текст статуса для экрана. |

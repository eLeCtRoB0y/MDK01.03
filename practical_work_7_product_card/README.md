# Практическая работа №7 — Карточка товара на Flutter

Реализация простого интерфейса карточки товара (электросамоката) в рамках учебного задания. Код написан без использования сложных кастомных виджетов, чтобы строго соответствовать требованиям лабораторной работы.

## Выполненные требования ТЗ
* **Базовый контейнер**: Весь контент обернут в `Container` с черной рамкой без скруглений (`Border.all()`).
* **Основная разметка**: Все элементы выровнены по вертикали внутри единого виджета `Column`.
* **Заголовок**: Название товара выведено крупным жирным текстом.
* **Изображение**: Картинка самоката загружается напрямую из сети через `Image.network`.
* **Описание**: Характеристики устройства помещены в отдельный `Container` с внутренними отступами (`padding`).
* **Строка параметров**: Цена и рейтинг товара выстроены горизонтально в один ряд с помощью `Row`.

## Особенности учебного кода
* Для демонстрации ручного управления направлением текста у каждого текстового элемента сохранен параметр `textDirection: TextDirection.ltr`.
* Для стабильной работы и корректного отображения шрифтов на эмуляторе проект обернут в базовые виджеты `MaterialApp` и `Scaffold`.
* Задано ограничение `mainAxisSize: MainAxisSize.min`, благодаря которому карточка не растягивается на весь экран и не ломает верстку.

## Инструкция по запуску
1. Скопируйте код из файла `main.dart`.
2. Убедитесь, что ваш эмулятор или устройство имеет доступ к интернету для загрузки изображения.
3. Запустите проект клавишей **F5** в Android Studio/VS Code.
